# Laboratorio de Git

Este repositorio contiene las prácticas realizadas para el laboratorio de comandos básicos de Git. A continuación, se describen los pasos realizados:

1. **Creación del repositorio local:** Inicializamos el repositorio y añadimos un archivo README.md.
2. **Subida a GitHub:** Conectamos el repositorio local con GitHub y subimos los cambios iniciales.
3. **Commit y Push:** Hicimos un commit y subimos los cambios a GitHub.
4. **Creación de la rama 'development':** Creamos una nueva rama y realizamos cambios en ella.
5. **Merge de ramas:** Realizamos el merge de la rama 'development' con la rama principal.
